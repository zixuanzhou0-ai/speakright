## 产品简介

SpeakRight 是一款专为中文学习者打造的发音训练桌面端。当前英语是稳定基线，西班牙语、法语、俄语是实验板块；你可以在设置里切换学习语言。

核心功能包括：

- **听当前语言标准示范** — 由 TTS 或本地发音包提供示范朗读
- **录音跟读** — 在浏览器中直接录音，无需安装任何软件
- **AI 发音评分** — 基于微软 Azure Speech 服务，从准确度、流利度、完整度、韵律四个维度精准评分
- **AI 中文教练反馈** — 大语言模型根据评分结果生成详细的中文发音指导，告诉你具体哪里需要改进
- **40 个音标练习** — 覆盖 16 个元音（含双元音）和 24 个辅音，每个音标配有多个示例单词
- **自由练习** — 输入任意英文句子进行跟读练习，支持逐词评分和韵律分析

## 首次使用配置

点击导航栏右上角的「设置」进入设置页面。你需要配置以下 API 密钥才能使用完整功能。

**所有密钥仅保存在你的浏览器本地（localStorage），绝不会上传到任何服务器。**

### 1. Azure Speech（语音评分）

**用途：** 对你的录音发音进行 AI 评分，提供准确度、流利度、完整度和韵律四个维度的分数。这是核心评分功能，**必须配置**。

**获取方式：**

1. 访问 [Azure 门户](https://portal.azure.com)，使用微软账户登录
2. 在搜索栏搜索「语音服务」（Speech Services）
3. 点击「创建」，选择 **F0 免费层**（Free tier）
4. 创建完成后，进入该资源的「密钥和终结点」页面
5. 复制 **密钥 1**（Key 1）和 **区域**（如 `eastus`、`eastasia`）

**免费额度：** F0 免费层每月提供 5 小时语音识别时长（约 18,000 秒），足够日常练习使用。

**配置步骤：**

1. 在设置页找到「Azure Speech」卡片
2. 填入 Subscription Key（密钥）
3. 填入 Region（区域，如 `eastus`）
4. 点击「测试连接」按钮验证配置是否正确

### 2. ElevenLabs（标准发音示范）

**用途：** 在自由练习页生成当前语言的标准示范朗读，提供 TTS 语音合成服务。单词词典发音是另一套服务（见下方），不需要此配置。

**获取方式：**

1. 访问 [elevenlabs.io](https://elevenlabs.io) 注册账户
2. 登录后进入 Profile 页面
3. 在 API Key 部分获取你的密钥

**免费额度：** 每月约 10,000 字符（大约可以合成 100 次句子发音）。

**配置步骤：**

1. 在设置页找到「ElevenLabs」卡片
2. 填入 API Key
3. 选择你喜欢的语音（可选）：
   - **Eryn** / **Daphne** / **Nichalia** / **Liz** — 女声
   - **Brian** / **Micheal Scott** / **Max** — 男声
4. 选择模型（推荐 `eleven_flash_v2_5`，速度快且质量好）
5. 点击「测试连接」验证

### 3. 发音音源（单词发音）

**用途：** 音标详情页中示例单词的发音来源。

提供两种音源供选择：

| 音源 | 特点 | 是否需要配置 |
|------|------|-------------|
| **有道词典**（默认） | 国内访问速度快，发音质量好 | 无需配置，开箱即用 |
| **韦氏词典** | 权威美式发音，海外访问更快 | 需要 API Key |

**韦氏词典 API Key 获取方式：**

1. 访问 [dictionaryapi.com](https://dictionaryapi.com) 注册开发者账户
2. 申请 Collegiate Dictionary API Key
3. 在设置页选择「韦氏词典」后填入 API Key

**兜底机制：** 如果在线词典不可用，App 会自动使用本地预录的音频文件播放发音。

### 4. LLM 大模型（AI 教练反馈）

**用途：** 根据 Azure 评分结果，生成详细的中文发音指导反馈。AI 教练会指出你的具体发音问题，并给出改进建议。

**支持的提供商（均使用 OpenAI 兼容接口）：**

| 提供商 | Base URL | 推荐场景 |
|--------|----------|---------|
| **DeepSeek** | api.deepseek.com | 性价比极高，推荐首选 |
| **Qwen / 通义千问** | dashscope.aliyuncs.com | 国内访问快，阿里出品 |
| **GLM / 智谱** | open.bigmodel.cn | 国产大模型，中文能力强 |
| **Kimi** | api.moonshot.cn | Moonshot 出品 |
| **Doubao / 豆包** | ark.cn-beijing.volces.com | 字节跳动出品 |
| **Claude** | api.anthropic.com | Anthropic 出品 |
| **GPT** | api.openai.com | OpenAI 出品 |
| **Gemini** | generativelanguage.googleapis.com | Google 出品 |
| **自定义** | 任意 URL | 填写任意 OpenAI 兼容 API 地址 |

**配置步骤：**

1. 在设置页找到「LLM 大模型」卡片
2. 选择提供商（Base URL 会自动填充）
3. 填入 API Key
4. 选择或输入模型名称（可从预设 chips 中快速选择，也可自由输入）
5. 点击「测试连接」验证

**推荐配置：** DeepSeek（价格便宜、中文质量好）或通义千问（国内访问快、免费额度多）。

## 音标练习

### 浏览音标列表

1. 点击导航栏「音标」进入音标列表页
2. 页面顶部有两个标签页：
   - **元音**（16 个）— 包含单元音和双元音
   - **辅音**（24 个）
3. 每张音标卡片显示以下信息：
   - **IPA 符号**（大字显示）
   - **难度标签** — 简单（绿色）/ 中等（黄色）/ 困难（红色）
   - **中文发音描述** — 简明扼要的发音要领说明
   - **示例单词** — 含 IPA 标注和 Emoji 图标
   - **播放按钮** — 点击可试听该音标的标准发音
4. 点击卡片任意位置（播放按钮除外）进入该音标的详情练习页

### 音标详情页练习流程

进入详情页后，桌面端为左右双栏布局，移动端为上下单栏布局。

#### 第一步：了解发音方式

- **左栏上方** 显示 IPA 符号和口型示范视频
- 观看视频了解嘴唇、舌头的正确位置

#### 第二步：听标准发音

- 页面展示当前练习单词（附 IPA 音标标注）
- 点击单词旁的播放按钮，听标准发音
- 播放时单词会高亮放大，配合涟漪脉冲动画

#### 第三步：切换练习单词

- 每个音标有约 20 个练习单词（8 个核心词 + 12 个扩展词）
- 点击 **右箭头** → 随机切换到新单词（智能选词：未练过的词优先出现）
- 点击 **左箭头** → 回退到上一个练习过的词
- 顶部显示「已练 X/Y」进度计数器

#### 第四步：录音跟读

1. 点击中央的 **录音按钮**（青色渐变圆形按钮）开始录音
2. 录音过程中：
   - 显示倒计时进度条（最长 30 秒）
   - 最后 5 秒进度条变为红色，提醒即将超时
   - 超时后自动停止录音并触发评分
3. 说完后点击录音按钮停止录音

#### 第五步：录音操作

录音完成后，下方显示 3 个圆形操作按钮：

- **回放** — 重听自己刚才的录音
- **清除** — 删除当前录音，重新来过
- **评分** — 将录音提交给 Azure Speech 进行 AI 评分

#### 第六步：查看评分结果

**左栏 — 分数概览：**

- **总分方块** — 大字显示总分（0-100）
  - 80 分以上：青色背景（优秀）
  - 60-79 分：黄色背景（良好）
  - 60 分以下：红色背景（需要加强）
- **四维分数条** — 准确度、流利度、完整度各有独立进度条
- **趋势图** — 显示最近 5 次评分的趋势折线图

**右栏上方 — 音素分析：**

- 每个音素独立标注颜色（青色=好，黄色=一般，红色=差）
- 点击任意音素可查看详细评分信息

**右栏下方 — AI 教练反馈：**

- 中文反馈内容以流式方式逐字显示
- 内容包括：整体评价、具体问题指出、改进建议
- 教练风格直接严格，不客套，专注于帮你发现问题

## 自由练习

点击导航栏「句子」进入自由练习页。

#### 第一步：输入英文句子

- 在文本框中输入你想练习的英文句子
- **字符限制：150 字符**，右下角显示实时字符计数
- 接近上限时计数器变为警告色

#### 第二步：听标准发音

- 点击文本框左侧的 **大播放按钮**（渐变色方块），播放 TTS 标准发音
- 播放时文字会 **卡拉 OK 式逐词高亮**，帮助你跟上节奏
- 下方 **速度滑块** 可调节播放速度（0.7x 慢速 ~ 1.2x 快速，步进 0.1）
- 播放完毕后可点击 **「重听一遍」** 按钮重播（使用缓存，不消耗 API 额度）

#### 第三步：录音跟读

操作方式同音标练习：

1. 点击录音按钮开始录音（最长 30 秒）
2. 清晰朗读整个句子
3. 点击停止或等待超时自动停止

#### 第四步：查看评分结果

**左栏 — 分数概览：**

- 总分 + 准确度/流利度/完整度/**韵律**（句子模式额外显示韵律评分）
- 韵律评分反映你的语调、重音和停顿是否自然

**右栏上方 — 逐词评分：**

- 每个单词用颜色标注评分（青色=好，黄色=一般，红色=差）
- 一眼看出哪些单词发音有问题

**右栏下方 — AI 教练反馈：**

- 针对句子的整体发音给出中文反馈
- 包含语调、连读、重音等句子层面的发音指导

#### 单词模式

当你在自由练习页只输入一个单词时，页面会自动切换为单词模式：

- 发音来源自动切换为有道词典或韦氏词典（根据设置页的发音音源配置）
- 显示区域会展示单词和 IPA 音标（如有）
- 语速调节和卡拉 OK 逐词高亮自动隐藏（单词无需这些功能）
- AI 教练反馈会针对单词的每个音素给出更详细的发音指导
- 输入多个单词后自动切回句子模式，无需手动切换

## 用量监控

设置页顶部显示三项 API 用量统计，帮助你掌握免费额度的使用情况：

| 监控项 | 数据来源 | 说明 |
|--------|---------|------|
| **ElevenLabs** | API 实时查询 | 显示已用字符数 / 月度上限，每次 TTS 调用后自动刷新 |
| **Azure Speech** | 本地估算 | 显示已用时长（秒），基于本地录音时长累计，可手动重置 |
| **LLM 大模型** | 本地记录 | 显示已消耗 Token 数 + 估算费用（人民币） |

## 主题切换

- 点击导航栏右上角的 **太阳/月亮图标** 切换深色模式和浅色模式
- 设置会自动保存到浏览器，下次打开时保持上次的主题选择
- 品牌主色为青色（Teal），在两种主题下均有良好的视觉效果

## 数据存储说明

SpeakRight 当前版本的所有数据均存储在浏览器本地：

| 数据类型 | 存储位置 | 说明 |
|---------|---------|------|
| API 密钥配置 | localStorage | 所有 API Key 和设置项 |
| 练习历史记录 | localStorage | 评分记录、练习进度 |
| 用量统计 | localStorage | Azure 时长、LLM Token 消耗 |
| TTS 音频缓存 | IndexedDB | 最多保存 50 条，LRU 淘汰策略 |

**重要提醒：**

- **清除浏览器数据**（清除缓存、Cookie、站点数据）**会导致所有配置和历史记录永久丢失**
- 建议单独记录保存你的 API Key 等重要配置信息
- 更换浏览器或设备后需要重新配置所有设置
- 使用浏览器的「无痕/隐私模式」时，关闭窗口后数据即丢失

## 常见问题 FAQ

### 评分相关

**Q: 为什么我的评分总是很低？**

A: 请尝试以下方法改善：
- 在安静的环境中录音，避免背景噪音
- 使用质量较好的麦克风（耳机麦克风通常比笔记本内置麦克风效果好）
- 录音时尽量清晰、完整地发音，不要太快
- 确保麦克风音量适中，不要离嘴太近或太远

**Q: 评分时一直转圈/报错怎么办？**

A: 检查以下几点：
- Azure Speech 配置是否正确（设置页点击「测试连接」验证）
- 网络连接是否正常（Azure 服务需要联网）
- 录音是否太短（过短的录音可能无法评分）

### 发音播放相关

**Q: 为什么听不到标准发音？**

A: 需要区分两种场景：
- **单词发音**（音标详情页）：使用有道/韦氏词典，检查网络连接。如在线不可用，会自动使用本地预录音频
- **句子发音**（自由练习页）：需要配置 ElevenLabs API Key，在设置页检查是否配置正确并通过测试连接

**Q: 句子发音播放很慢/没有声音？**

A: 首次播放需要调用 API 合成语音，可能需要几秒钟。合成完成后会缓存到本地，后续重播无延迟。检查 ElevenLabs 免费额度是否用完。

### AI 反馈相关

**Q: AI 反馈没有出现怎么办？**

A: 检查以下几点：
- LLM 配置是否正确（提供商、API Key、Base URL、Model 都要填写正确）
- 点击「测试连接」验证配置
- 推荐使用 DeepSeek 或通义千问，国内访问快且价格低

**Q: AI 反馈内容质量不好？**

A: 可以尝试更换模型。不同大模型对中文发音指导的质量有差异，一般来说参数量更大的模型效果更好。

### 额度与费用

**Q: 免费额度够用吗？**

A: 对于日常练习完全够用：
- **Azure F0**：每月 5 小时（约 18,000 秒），按每次评分 5 秒计算，约可评分 3,600 次
- **ElevenLabs 免费版**：每月约 10,000 字符，约可合成 100 次句子发音
- **LLM**：费用取决于提供商。DeepSeek 非常便宜，每次反馈约几分钱人民币

**Q: 如何节省 ElevenLabs 额度？**

A: 多使用「重听一遍」功能重播已缓存的语音，不会消耗额度。只有首次合成新句子时才消耗字符配额。

### 设备与浏览器

**Q: 换浏览器或设备后数据还在吗？**

A: 不在。所有数据保存在当前浏览器的本地存储中。更换浏览器或设备后需要重新配置所有 API Key 和设置。

**Q: 支持手机使用吗？**

A: 支持。App 采用响应式设计，手机端自动切换为单栏布局。需要注意：
- 浏览器需要支持并授权麦克风权限
- 推荐使用 Chrome 或 Safari 浏览器
- 首次录音时浏览器会弹出麦克风授权请求，请点击「允许」

**Q: 支持哪些浏览器？**

A: 推荐使用最新版的 Chrome、Edge、Safari 或 Firefox。需要浏览器支持 MediaRecorder API（用于录音）和 Web Audio API（用于音频播放）。

**Q: 音节上方的 ˈ 和 ˌ 符号是什么意思？**

A: ˈ 表示主重音（primary stress），标记在需要重读的音节前面；ˌ 表示次重音（secondary stress）。例如 "afternoon" 的音节标注为 ˌ/æf/ · /tər/ · ˈ/nuːn/，说明第三个音节 /nuːn/ 是主重音，第一个音节 /æf/ 是次重音。单音节词不显示重音标记。

**Q: 页面切换后之前的练习记录还在吗？**

A: 在同一个浏览器标签页内，切换音标练习和自由练习页面时，之前的输入文本、评分结果和 AI 反馈会自动保留。关闭标签页后数据会自动清除。如需手动清除，点击页面右上角的"清除练习记录"按钮。
